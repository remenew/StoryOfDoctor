**检查日期**: 2026-03-31

## 一、发现的不一致性

### 1. 卡牌类型定义不一致 ⚠️

**设计文档定义** (第3.2节):
| 类型 | 图标 | 用途 |
|------|------|------|
| **草药卡** | 🌿 | 治疗基础，组成药方 |
| **药方卡** | 📜 | 组合技触发，治疗效果加成 |
| **技能卡** | ⚡ | 战斗中使用，特殊效果 |
| **任务卡** | 📋 | 接取任务，推进剧情 |

**实际实现** (`cards.json`):
| 类型 | 数量 | 说明 |
|------|------|------|
| `drug` | ~830张 | 药物牌（对应设计文档的"草药卡"） |
| `magic` | 4张 | 巫术牌（祝由术、符咒、咒水、气鼓） |
| `secret` | 12张 | 秘方牌（银翘散、麻黄汤等方剂） |

**问题**:
1. 类型名不一致：`drug` vs `herb`
2. 缺少独立的"技能卡"类型（技能已移至独立系统）
3. 缺少"任务卡"类型
4. `magic` 和 `secret` 类型在设计文档中归类不明确

**建议**:
- 统一类型命名：`drug` → `herb`
- 明确 `magic` 和 `secret` 的定位

### 2. 攻击力/治疗值计算不一致 ⚠️

**设计文档定义** (第5.5节):
```
草药牌攻击值：
君（×4）= 20    ——  决定治疗方向
臣（×3）= 15    ——  加强主药
佐（×2）= 10    ——  调和配伍
使（×1）= 5     ——  引经，基础

随机草药牌（非对症方剂草药，占战斗牌池约 30%）：
攻击值：随机整数 -5 ~ +8（负值 = 配伍禁忌，误用反伤）
```

**实际实现**:
```javascript
// DiseaseCardSystem.js
const ROLE_ATTACK_POWER = {
  '君': 20,
  '臣': 15,
  '佐': 10,
  '使': 5
};

// 随机牌攻击力范围
randomAttackMin: -10,
randomAttackMax: 10
```

**不一致点**:
1. 随机牌范围：设计 `-5 ~ +8` vs 实现 `-10 ~ +10`

**状态**: ✅ 已按最新需求调整为 `-10 ~ +10`

### 4. 数据结构不一致 ⚠️

**设计文档定义** (第8节):
```javascript
// diseases.json 结构
diseases: [
  {
    id: "lung_0000",
    name: "感冒风寒束表证",
    formulaRef: "荆防败毒散",
    difficulty: "一般",
    attackBase: 20,
    attackVariance: [0.8, 1.2],
    ultimateAttack: { ... }
  }
]
```

**实际实现**:
```javascript
// 已实现合并的 diseases.json 结构
diseases: [
  {
    id: "lung_0000",
    name: "感冒风寒束表证",
    formula: "荆防败毒散",
    difficulty: "normal",
    attackDeck: [...],  // 来自 disease-attack.json
    effectiveHerbs: [...],  // 来自 disease-cards.json
    totalAttackPower: 0
  }
]
```

**不一致点**:
1. `formulaRef` vs `formula`
2. `difficulty` 值：设计用中文"一般" vs 实现用英文"normal"
3. 缺少 `attackBase` 和 `attackVariance` 字段
4. 使用 `attackDeck` 数组而不是 `ultimateAttack` 对象

## 二、已修复的问题

### 1. 疾病-草药关联数据合并 ✅

**问题**: `disease-cards.json` 和 `diseases.json` 内容重复

**解决方案**: 已将 `disease-cards.json` 合并到 `diseases.json`
- 添加了 `effectiveHerbs` 字段
- 添加了 `totalAttackPower` 字段
- 在 `meta` 中添加了 `effectiveHerbsMeta`

### 2. 技能牌从卡牌中移除 ✅

**问题**: 技能牌不应该在草药牌组中

**解决方案**: 已从 `cards.json` 中移除所有技能牌（针灸、推拿、拔罐、气功、艾灸、刮痧）

### 3. 攻击力概念澄清 ✅

**问题**: 攻击力是独立属性还是治疗加成？

**解决方案**: 
- 攻击力 = 治疗加成值（不是百分比）
- 有效药物：基础治疗值 + 攻击力
- 非有效药物：只有攻击力（可正可负）
